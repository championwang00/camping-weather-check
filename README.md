# 露营天气检查

这是一个给露营出发前用的 Codex Skill。它不只回答“明天晴不晴”，而是把天气拆成露营现场真正会遇到的几个问题：帐篷什么时候搭、饭点会不会下雨、夜里会不会冷醒、天幕扛不扛得住风、日落和星空还有没有戏。

## 它怎么判断

核心是“营地五步天气判断法”：

1. **查精确位置**  
   不拿城市天气糊弄自己。山里、溪边、湖边、海边的天气，经常和城区不是一回事。

2. **查降雨**  
   不只看有雨没雨，要看几点下、下多久、雨量多大，是不是刚好撞上搭帐篷、做饭、睡觉或撤营。

3. **查温度**  
   重点看夜间最低温。露营舒服不舒服，很多时候不是白天热不热，而是凌晨两三点会不会被冷醒。

4. **查风速**  
   4 级以内通常还能轻松露营；5 级开始要认真防风；6 级以上，帐篷和天幕就不要硬扛了。

5. **查云量**  
   想看日落、星空、日出，就不能只看降雨。云厚、低云、起雾，即使没下雨，也可能让这一趟少一半意思。

## 它会给什么结果

每次检查会给出三档结论：

- **适合**：天气总体友好，常规装备即可。
- **谨慎**：能去，但要调整营位、装备、行程或撤营时间。
- **不建议**：雷暴、强风、暴雨、低温或地形风险明显，建议延期或换地点。

同时会补充这些安全项：

- 雷暴、闪电、暴雨、大风等预警
- 山谷、溪边、湖边、森林、海边等地形风险
- 上游降雨和水边营地风险
- 夜间低温和睡袋温标匹配
- 撤营时间和撤离路线

## 适合什么场景

- 出发前问：“这个周末适不适合露营？”
- 临时改地点前问：“这个营地天气稳不稳？”
- 亲子露营前做保守判断
- 山地、溪边、湖边营地出发前复查
- 想看日落星空时判断云量和体验
